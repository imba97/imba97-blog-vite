# 关于 ID

大概是初中的时候，那时候在对战平台玩《魔兽争霸3》，就想给自己起个网名。

想来想去，想到我看的一个电子竞技的电视节目：游戏风云，论坛的水友ID都是以imba开头，所以这是我ID里“imba”的来历，又因为我是1997年的，所以是“久期”，当时在想用哪个 jiu 哪个 qi，因为我性格，是个对于人、事只会期盼、期待不会主动的人，所以最后定为“久期”。
